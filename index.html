Customer Churn Prediction

In this project I have implemented a customer churn prediction application, that can predict if a customer will churn or not. The model is based on the customer's past behavior and the current customer's behavior. The model is trained on a dataset of 100k customers. The model is tested on a dataset of 10k customers. The model is deployed on Heroku.
